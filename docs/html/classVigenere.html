Detailed Description

File: vigenere.h

File that outlines the Vigenere class. The Vigenere class uses a set key to encrypt and decrypt messages, with the letters of the key acting as guidelines to move the letters in the original message a certain amount.

For example: 'A' => no movement, 'D' => Move three letters forward.

The current implementation of this function retains the whitespace it is provided in the input. It also assumes all inputs and outputs will be in uppercase.
